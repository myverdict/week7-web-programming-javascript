  • Callback functions & Arrow notations (=>):
    • Animals : forEach() looping with a classic callback function & callback function using =>.
    • Button : button with an addEventListener() for a click event with a callback function
    • setTimeout(handler/fn, ms): a method that calls a function or evaluates an expression after a specified number of milliseconds.
      • Classic callback function : displays an alert after 3 seconds
      • Callback function with arrow notation (=>) : displays an alert after 3 seconds
    • setInterval(handler/fn, ms): a method that calls a function or evaluates an expression at specified intervals. It will continue calling the function until clearInterval() is called, or the window is closed.
      • Classic callback function : shows current date & time with seconds updating
      • Callback function with arrow notation (=>) : shows current date & time with seconds updating
    • Draw text on a picture : drawing an image on a canvas with text writing input features. Demonstrates the load event listener with a callback function.
